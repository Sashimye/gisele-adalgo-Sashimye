# 🧠 Adalgo

Projet individuel (1 semaine) — **Ada Tech School**  
Apprentissage des **tests unitaires** en JavaScript avec **Vitest**.

📂 Repo : `adalgo-sammykelly`

***

## ⚙️ Installation

```bash
git clone https://github.com/sammykelly/gisele-adalgo-Sashimye.git
cd gisele-adalgo-Sashimye
pnpm install
pnpm test
```

***

## 🧩 Fonctionnalités

- ✅ Écriture de **tests unitaires** avec Vitest
- ✅ Implémentation de fonctions JS (algorithmes, tableaux, objets, callbacks)
- ✅ Respect du format `n-nomDeLaFonction.js` / `n-nomDeLaFonction.test.js`
- ✅ Exercices progressifs : `1-chocolatine.js` → `6-*` + bonus

***

## 🛠️ Stack

**JavaScript (ES6+)** -  **Node.js** -  **Vitest** -  **pnpm**

***

## 🎓 Compétences

- Algorithmique de base (conditions, boucles, tableaux)
- Écriture et exécution de tests unitaires
- Compréhension du **TDD** (Test-Driven Development)
- Maîtrise des callbacks, objets et méthodes JS natives
